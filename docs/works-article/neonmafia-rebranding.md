# NEONMAFIA株式会社リブランディング

Year: 2021年
Period: 3ヶ月
Overview: 所属会社の関連企業で動画を中心としたクリエイティブ制作を手掛けるNEONMAFIA株式会社の事業戦略策定・ブランドサイトデザインを行うプロジェクト。
Type: リブランディング
Roles: ディレクター, デザイナー, リサーチャー
Actions: マーケットリサーチ、競合調査の結果、他のクリエイティブ制作会社と比べインフルエンサーを活用したマーケティングが強みであることがわかりました。また、包括的にクリエイティブを提供するという強みを活かすため、ブランドの方向性を「動画制作に強い制作会社」から「ブランディングを軸としたクリエイティブブティック」へと転換。クライアントが付加価値を感じられるようなコピーライティング、サービス資料のデザインを制作しました
Webサイトはインバウンドマーケティングツール + ポートフォリオとして機能するように、ユーザーがランディング→実績→問い合わせまでスムーズに回遊できるような設計、実績がより魅力的に見えるようなビジュアルデザインを心がけました。
NDA Safe: No
Problem (As-Is): NEONMAFIA株式会社では動画以外にもブランディングコンサルティングやWebサイト制作、イベント企画・制作などクリエイティブに関わるサービスを包括的に提供していましたが、動画制作会社としての認知が強く、新たな販路開拓に課題を持っていました。
Scope: Webデザイン, ビジュアルデザイン, リブランディング, 情報設計, 要件定義
Skills: BX, IA設計, UIデザイン, UXデザイン, Webデザイン, アートディレクション, コミュニケーションデザイン, ディレクション, ライティング, リサーチ
Team Size: 2〜3
Tools/Technology: Adobe XD
