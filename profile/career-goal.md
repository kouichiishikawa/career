# キャリアゴール

## 1. 現在地とキャリアの転換点

### 現在の役割と経験の概要

株式会社スターミュージック・エンタテインメントにて、デザインマネージャーとして幅広い領域を担当しています。UI/UXデザインを軸としながら、コーポレートブランディング、デザインシステム構築、システム開発、DX推進、チームマネジメントまで、事業横断で戦略から実装・運用まで伴走してきました。

**主な実績：**
- デザインシステム「Interstellar」の0→1構築により、デザイン工数を50%削減
- コーポレートサイトリニューアルで平均セッション時間を111%改善（1:17→2:43）
- AIコーディングを活用した高速Web制作（1ヶ月でCVR 2.1%→3.6%を達成）
- Culture Deck制作により選考エントリー率を48%向上（2.7%→4.0%）

### キャリア転換を考えるきっかけ

これまでのキャリアでは、**「作って終わり」のプロジェクト**がほとんどでした。プロジェクト完了後の継続的な改善に関わる機会が限られていたため、デザインの本当の価値を発揮しきれていないという課題を感じていました。

転機となったのは、業務時間外の課外プロジェクトとして取り組んだデザインシステム「Interstellar」の構築・運用です。通常の業務プロジェクトとは異なり、納品期限に縛られずに継続的な改善に時間を割けたことで、実際のプロジェクトで使われるたびにフィードバックを受け、コンポーネントを改善し、ドキュメントを充実させることができました。この改善サイクルを通じて組織全体のデザイン品質が向上していく過程を経験し、**継続的な改善と進化が組織に価値を生み続ける**ことを実感しました。デザインが一度作って終わりではなく、プロダクトと共に成長し続けることの重要性と面白さを学び、**デザインが継続的に価値を生み出すサイクル**に深く関わりたいという思いが強くなりました。

### これまでの実績から得た学び

**① デザインシステムが組織を変える**
デザインシステム「Interstellar」の構築を通じて、デザインが単なる見た目ではなく、組織のコミュニケーション基盤であり、意思決定の質とスピードを変える力を持つことを実感しました。

**② AIとの協働が生産性を飛躍させる**
Figma MCP + Claude Codeを活用したAIコーディングにより、デザインから実装までのリードタイムを劇的に短縮。デザイナーが実装まで関与することで、UXの質を担保しながら高速でイテレーションを回せることを証明しました。

**③ ビジネス視点とデザインの統合**
経営層に近い立場で仕事をした経験から、ビジネスゴールとユーザー体験を両立させる重要性を学びました。CVR改善、セッション時間向上、検索順位向上など、定量的な成果で価値を示すことの重要性を理解しています。

## 2. 目指すキャリアの方向性

### 主軸：デジタルプロダクトのUI/UXデザイナー

**デジタルプロダクト**の領域で、ユーザー体験の設計から実装、改善までを一貫して担えるUI/UXデザイナーを目指します。SaaSやWebアプリケーションなど、継続的に改善され成長するプロダクトに深く関わり、**仮説→実装→検証→改善**のサイクルを高速で回すことで、ユーザーとビジネスの双方に価値を提供したいと考えています。

**具体的に経験したいこと：**
- 定量的なKPI改善（CVR、リテンション、NPS等）
- ユーザーフィードバックを活かしたプロダクト改善
- A/Bテストや仮説検証のサイクル
- デザインシステムの構築・運用とプロダクトへの接続

### サブ軸①：デザインマネージャー・デザインリード

プレイヤーとして自らデザインに手を動かしながら、チームをリードし、デザイン組織全体の価値を最大化する役割にも挑戦したいと考えています。デザインプロセスの最適化、デザインシステムの浸透、他職種との協働を推進し、デザインが事業成長のドライバーとなる環境を作りたいです。

**目指す姿：**
- デザインリード：プロダクトのデザイン戦略をリードし、UX品質を担保
- デザインマネージャー：チームのマネジメントとデザイン組織の成長を推進
- CDO（CXO）：経営視点でデザインの価値を最大化

### サブ軸②：BXデザイン・コミュニケーションデザイン

プロダクトUIだけでなく、ブランド体験全体の設計にも強い関心があります。コーポレートサイト、Culture Deck、デザインシステムなど、複数のタッチポイントでブランドの一貫性を保ちながら、ユーザーに価値を届けた経験を活かし、プロダクトとブランドを統合的にデザインしたいと考えています。

### 重視する働き方・環境

**① カルチャーフィット（最優先）**
- 透明性の高いコミュニケーション
- デザイナーが意思決定に関われる環境
- お互いにリスペクトし合えるチーム文化
- ボトムアップの意思決定プロセス

**② 業務内容**
- 「作って終わり」ではなく、改善・グロースを経験できる
- デザインが事業成長に直結する環境
- デザインシステムの構築・運用に関われる

**③ 年収・待遇**
- 適正な市場価値に基づく評価

**④ 成長機会**
- デザインスキルだけでなく、ビジネス・技術の両面で成長できる
- デザイン組織の立ち上げ・拡大に関われる

**⑤ 働きやすさ**
- ハイブリッドワーク可能（週2〜3日リモートが理想）
- ライフワークバランスを保てる環境

### 避けたい環境

- デザイナーが意思決定に関われない環境
- トップダウンで指示だけが降りてくる組織
- 情報やコミュニケーションの透明性が低い
- チーム文化が合わない（リスペクトの欠如、心理的安全性の低さ）
- 「作って終わり」のプロジェクト型業務

## 3. なぜその方向を目指すのか

### ① 「作って終わり」から「改善・グロース」へ

これまでのキャリアでは、デザインを納品した後の改善サイクルに関われないことが多く、もどかしさを感じていました。デザインの真価は、**継続的な改善を通じてユーザーとビジネスに価値を提供し続けること**にあると考えています。

デザインシステム「Interstellar」の構築・運用を通じて、この考えが確信に変わりました。実際のプロジェクトで使われるたびにフィードバックを受け、コンポーネントを改善し、ドキュメントを充実させることで、組織全体のデザイン品質が向上していく過程を経験しました。この継続的な改善サイクルが組織に価値を生み続けることを実感し、今後はプロダクトの成長を数字で追いかけながら、デザインの価値を証明し続けたいと考えています。

### ② 世界から評価されるプロダクト/UX/ブランドを創りたい

日本国内だけでなく、**世界から評価されるプロダクトやUX、ブランドを生み出す仕事を一部でも担いたい**という思いがあります。そのためには、グローバルに通用するデザイン品質、アクセシビリティ、ユーザビリティを追求し、デザインシステムを基盤とした一貫性のある体験を提供する必要があります。

デザインシステム「Interstellar」の構築や、WCAG 2.1 AA準拠率96%を実現したOMYサイトリニューアルの経験を活かし、世界基準のデザイン品質を追求できる環境で挑戦したいと考えています。

### ③ ビジネス成長とユーザー価値を両立させたい

経営層に近い立場で仕事をした経験から、**デザインがビジネス成果に直結する重要性**を理解しています。美しいデザインを作ることだけが目的ではなく、ユーザーの課題を解決し、同時にビジネスの成長を後押しすることが、デザイナーの真の価値だと考えています。

今後は、CVR、リテンション、NPS、ARRなどのビジネスKPIを意識しながら、ユーザー体験を設計し、デザインの価値を定量的に証明できるデザイナーでありたいです。

### ④ ジェネラリストとしての強みを活かしたい

UI/UXデザインを軸としながら、ブランディング、マーケティング、エンジニアリング、ビジネス戦略など、幅広い領域での実務経験があります。この**ジェネラリストとしての強み**を活かし、デザインだけでなく、事業全体を俯瞰しながら価値を提供できる環境で働きたいと考えています。

特にスタートアップのような、デザイナーが複数の役割を担いながら事業成長に貢献できる環境にフィットすると考えています。

## 4. 実現するための強み

### ① デザインシステム構築・運用の実績

デザインシステム「Interstellar」を0→1で構築し、実際のプロジェクトで活用することで、デザイン工数を50%削減、ブランドの一貫性を実現しました。カラーパレット104色すべてでWCAG 2.1 AA準拠、デザイントークンのJSON化、コンポーネントライブラリの整備など、デザインと開発を接続する仕組みを構築した経験があります。

**再現可能なスキル：**
- カラーシステム設計（アクセシビリティ対応）
- デザイントークン設計（Figma Variables）
- コンポーネント設計（Atomic Design）
- デザインシステムのドキュメンテーション

### ② AIコーディングによる高速実装力

Figma MCP + Claude Code + Codexを活用し、デザインから実装までを一気通貫で担当。株式会社OMYのサイトリニューアルでは、稼働100時間（1ヶ月）で要件定義→デザイン→実装→公開までを完遂し、CVR 2.1%→3.6%を達成しました。

**技術スタック：**
- Next.js、React、Tailwind CSS、shadcn/ui
- Vercel、CI/CD（GitHub Actions）
- GA4、SendGrid
- Figma MCP、Claude Code、Codex

**価値：**
デザイナーが実装まで関与することで、デザイン意図を正確に反映し、UXの質を担保しながら高速でイテレーションを回せます。

### ③ ビジネス理解とデザイン戦略の統合

経営層に近い立場で仕事をした経験から、事業戦略とデザイン戦略を統合する力があります。単にUI/UXを整えるだけでなく、ビジネスゴール（CVR改善、セッション時間向上、検索順位向上等）を意識した設計ができます。

**実績例：**
- 株式会社OMY：CVR 2.1%→3.6%、検索順位15位→8.3位
- スターミュージックコーポレートサイト：平均セッション時間1:17→2:43（+111%）
- Culture Deck：選考エントリー率2.7%→4.0%（+48%）

### ④ ジェネラリストとしての幅広いスキル

UI/UXデザインを軸に、以下の領域で実務経験があります：

**デザイン領域：**
- UI/UXデザイン、情報アーキテクチャ、デザインシステム
- BXデザイン、コミュニケーションデザイン
- ビジュアルデザイン、プロトタイピング

**マーケティング・ビジネス領域：**
- SEO戦略、コンテンツマーケティング
- Web広告運用（リスティング、SNS広告）
- アクセス解析、データドリブンな意思決定

**エンジニアリング領域：**
- HTML/CSS、フロントエンド実装
- AIコーディング（Claude Code、Codex）
- CI/CD設計、パフォーマンス最適化

**マネジメント・ディレクション：**
- チームマネジメント、アートディレクション
- プロジェクトマネジメント、ステークホルダー調整
- 採用マーケティング、広報

この幅広いスキルセットにより、デザイナーが少ない組織でも即戦力として価値を提供できます。
